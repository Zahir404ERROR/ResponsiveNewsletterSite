Introduction

My website is a newsletter website that allows users to read interesting articles about  the latest and upcoming technology information. It allows people to download the weekly newsletter. The structure used is that of a standards newsletter; it has a slider which slides the featured articles.

Inspiration

The inspiration  for my website is a mixture of alternative tech newsletters like android authority and mit technology review.

The layout of the modals in the website were influenced by the MIT Technology Review structure.

The design was taken from a minimalist design by a youtube channel EasyTutorials.

Accessibility

ALT tags that show up when hovering over the image this allows people with bad eyesight who are unable to understand the image so they can read the image.

Header when the tab button is clicked it goes from each header to the next by clicking tab.

For people with bad vision, I ensured that the contrast ratio between text and its background is greater than 4.5 to 1 which is the standard arranged by WCAG.

Usability

To make the website Usable I made the buttons more interactive by increasing the size of the button when the mouse is hovering over it similarly when hovering over list items the color changes this makes the website more interactive.

The minimalist design doesn’t distract people in comparison with a crowded site. I made sure that the centre of the webpage ensures that all key information is viewable without scrolling.

I animated sliders to make them more interactive and smoothen the experience when the articles are changing.

Learning

Firstly, I had to learn how to make modals that popped up for each article as making a HTML page for each article is not very efficient programming to learn how to use modals. I learned it from w3schools. But what I wanted to achieve with modals was a bit more complicated than what they were asking so I read multiple Stack Overflow statements and problems other developers had when building the same type of modals. Doing this gave me a more comprehensive understanding for my implementation of modals.

I also had to learn how to implement sliders and learn transitions and animations to create a more interactive website. I also used W3Schools for this.

I also learned how to edit scrolling bars using the webkit framework which I was unfamiliar with previously.

Evaluation I 

I really enjoyed the sliders as building a slider of photos was quite easy but I struggled implementing the slider for articles. I also successfully built multiple modals in one page. This allowed me to create article pages without making the user open single html pages which would take longer for them to load up. This is also very inefficient. I am also proud of the animations and transitions implemented on the website.

Evaluation II

I actually wished I had used some extra js library but I didn't really understand how they worked till later on and I didnt have any left over time to implement them into my code. If I were to do this again I would like to have planned my time better.
